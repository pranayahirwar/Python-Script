Simple Python script that will copy data from your latest copied items or clipboard to a file name 'copydata.txt'

If file is running in 'w' - write mode. Where everytime you start this script this will overwrite data on the same file. So Keep or Copy that data before running that script, again.
If file is running in 'a' - write mode. New data will be added to new line.

Basic installation will be needed before using script will be
pip install pyperclip
